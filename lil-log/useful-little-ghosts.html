Useful little ghosts

systems project work yak shaving

Project management sounds like something that should wear a fleece vest and ask for a status update. But sometimes it is just taking a fog bank and giving it edges.

A bug list is not just chores. It is a map of where the product currently lies to the person using it. “This loads too slowly.” “This text cuts off.” “This logged my workout wrong.” Tiny betrayals, each one fixable.

The useful ghost does not need to be dramatic. It just points at the next true thing. Then the next. Then, if necessary, it hands you clippers, because the yak has entered the room again.
